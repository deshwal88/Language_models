Text Generator

'Text Generator' is an intelligent system which completes any incomplete sentence or text. It generates text of adjustable length according to the context given without using any database. It can also be used to predict just the next word given a context in the text box. The web app needs atleast 30 words as context to generate the provided outcome.

  • Run for one generates a word for the context provided.
  • Custom run generates as many words as entered by the user.
  • The app keeps on adding context each time you hit Submit.
  • To clear the previously added data hit Reset.

Also, here '<unk>' means a unique name and 'N' means any number.
