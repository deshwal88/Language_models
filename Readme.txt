"Check_requirements": checks the requirements for running the GUI and downloads data("simple-examples").

"Composer-run": contain the GUI which enables user to interact with the language model. It enables user to write or note 
things according to which suggestions come up in the label bar at the bottom of the window. The suggestions are 
sequential outputs from the language model that is running along side.

"Rnn model": is the notebook which shows the assembling of the rnn model step by step using LSTM cells. (exported as
'rnn_model')

"read_func": is the self-made module containing a set of functions to read the PTB data.

"simple-examples": contain the data from Pensymvania Tree Bank(PTB) which is necessary to run the model.
